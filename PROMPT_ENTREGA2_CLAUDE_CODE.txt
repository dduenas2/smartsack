## CONTEXTO DEL TRABAJO DE GRADO

Estoy desarrollando mi trabajo de grado del programa de Ingeniería de Software en la Institución Universitaria Politécnico Grancolombiano. El proyecto se llama "SmartSack" y son 3 entregas oficiales:

- Entrega 1 (semana 5): planteamiento, marco teórico, metodología — YA FUE ENTREGADA Y CALIFICADA con observaciones aplicadas
- Entrega 2 (semana 10): ESTA — debe documentar el AVANCE de implementación real
- Entrega 3 (semana 16): final con resultados de usuarios, conclusiones y sustentación

SmartSack es una plataforma web inteligente para plantas de fabricación de sacos de papel con 3 pilares:
1. Digital Twin con vistas operario/supervisor en tiempo real (WebSockets)
2. Motor de ML predictivo (XGBoost) para predicción de retrasos
3. Asistente conversacional con LangChain + Claude API + function calling

Stack: React 18 + Vite, Python 3.11 + FastAPI + SQLAlchemy, PostgreSQL 16, Redis 7, Docker Compose, Nginx, XGBoost, LangChain.
